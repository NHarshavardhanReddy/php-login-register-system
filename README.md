# 🔐 PHP Login & Registration System

A full-stack user authentication system built using PHP and MySQL. This project allows users to register, log in securely, access a protected dashboard, and log out using session-based authentication.

---

## 🚀 Features

- User Registration (Store user data in database)
- Secure Login Authentication
- Session Management
- Logout Functionality
- Error Handling & Validation
- Responsive UI Design

---

## 🛠️ Tech Stack

- Frontend: HTML, CSS, JavaScript  
- Backend: PHP  
- Database: MySQL  
- Server: XAMPP (Apache)
